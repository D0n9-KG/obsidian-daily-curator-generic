obsidian-daily-curator-generic

## Short description

Configurable Obsidian daily-note curator skill for turning messy daily captures into an evolving knowledge base.

## Long description

A generic agent skill for Obsidian vaults. It helps transform messy daily notes, quick captures, imported web clips, papers, videos, code snippets, ideas, tasks, and questions into daily summaries, knowledge cards, source notes, MOCs/maps, and an evolving vault architecture. It is safe by default, configurable through `.daily-curator/config.json`, and includes dry-run bootstrap, quick-capture preparation, scan helpers, examples, and release checks.
